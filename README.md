# Computer Architecture Labs

## Overview

This repository contains three lab projects developed for the Computer Architecture course at Instituto Superior Técnico.

The work focuses on low-level performance analysis, memory hierarchy, and processor optimization techniques.

## Labs

### Lab 1 – Pipeline Optimization
- Analysis of pipeline execution
- Data and control hazards
- Data forwarding techniques
- Loop unrolling and instruction-level optimization
- Performance evaluation (CPI, stalls, speedup)

### Lab 2 – TLB Cache Simulator
- Implementation of a two-level TLB (L1 and L2)
- Fully associative caches with LRU replacement policy
- Write-back memory policy and inclusion property
- Virtual-to-physical address translation and invalidation mechanisms

### Lab 3 – Cache Modeling and Optimization
- Experimental characterization of L1 and L2 caches
- Analysis of cache size, block size, and associativity
- Performance profiling of memory accesses
- Optimization techniques:
  - matrix transpose
  - blocked (tiled) matrix multiplication

## Key Concepts

- Memory hierarchy (TLB, L1, L2, DRAM)
- Cache behavior and locality
- Pipeline hazards and instruction scheduling
- Performance optimization and profiling

## Notes

- Focus on performance analysis using empirical measurements
- Combines theoretical concepts with low-level implementation
